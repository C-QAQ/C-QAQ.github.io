C.Stone Game

  • 3(与 0 等价) 不会对答案有影响，忽略
  • 有 1 和 2 : 1 + 2 = 3, pay->2 —–>结果为(a1 == 0 || a2 == 0)
  • 只有 1 : 三个[1 + 1 + 1 = 3, pay->(1 + 2)], 两个[1 + 1 = 2, pay->1] (2,3)能组合出大于1的任何数字
  • 只有 2 : [2 + 2 + 2 = 6, pay->(4 + 2)], [2 + 2 = 4, pay->4]

D.Fight against involution

  • n 个 [L, R], 先按照R升序排序，R相同按照L降序排序
  • 维护前 i 个人的最大L第 i 个人的成绩取此最大L时保证他的成绩>=前i - 1个人,与初始时选择R为他的成绩的w[i]相同，符合题目要求
